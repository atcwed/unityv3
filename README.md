
# Dungeon Arcanum 3D

Proyecto base para Unity 3D inspirado en el juego HTML Dungeon Arcanum.

## Características
- Generador de mazmorras
- Enemigos con IA simple
- Movimiento en tercera persona
- Sistema de niveles
- Preparado para GitHub

## Cómo usar

1. Instala Unity Hub
2. Crea un proyecto 3D
3. Copia la carpeta **Assets** dentro del proyecto
4. Abre la escena y asigna:
   - Prefab de piso
   - Prefab de muro
   - Player

## Controles

WASD / Flechas para moverte.

## Subir a GitHub

```bash
git init
git add .
git commit -m "Dungeon Arcanum 3D"
git branch -M main
git remote add origin https://github.com/TUUSUARIO/DungeonArcanum3D
git push -u origin main
```
